# FamilyApp (家族アプリ)

家族間で家計や生活情報をリアルタイムに共有するためのWebアプリケーション群です。
Firebase (Auth, Firestore, Hosting) を基盤とし、夫婦での利用に特化した設計を行っています。

## 📦 収録アプリ

### 1. 💰 立て替え家計簿 (\`/public/kakei.html\`)
夫婦間での日々の立て替えを記録し、一目で精算額がわかる家計簿アプリです。
- **特徴**: 夫・妻のどちらかが金額を入力すると反対側はロックされる（排他入力制御）。
- **機能**: 
  - リアルタイム同期
  - \`Sortable.js\` を用いた直感的なドラッグ＆ドロップによる並び替え
  - 「済」の一括トグル・削除
  - 結果のJSONエクスポート ＆ 画面のスクリーンショット保存 (\`html2canvas\`)

### 2. 🛒 底値帳 (\`/public/sokone.html\`)
よく買う肉・野菜・日用品などの最安値を記録・比較するためのアプリです。
- **特徴**: スーパーでの買い物中に「これ安い！」を即座に判断・登録できるUI。
- **機能**:
  - 内容量と価格から「最安値」をランキング形式で自動計算（王冠マーク表示）。
  - 下書き（登録モード）による入力途中のデータ保護。
  - スマートなグループ管理（アコーディオン式、ドラッグ＆ドロップによる手動並べ替え・名前順/更新順ソート機能）。
  - 各品目ごとの登録件数表示や「過去価格」を下位へ優先ソートする表示ロジック。
  - JSON形式のインポート・エクスポート機能。

## 💡 アプリ共通UIと追加開発
アプリをシームレスに行き来できるように、共通のナビゲーションメニュー (\`nav.js\`) を導入し、右上のハンバーガーメニュー（☰）から「トップ」「家計精算」「底値帳」などを瞬時に切り替えられるようにしています。
アプリを追加した場合は、この \`nav.js\` と \`index.html\` にリンクを定義するだけで統一したメニューを利用できます。

## 🛠 セットアップ
1. Firebase Console でプロジェクトを作成し、Firestore と Authentication を有効化。
2. \`/public/firebase-config.js\` を作成し、プロジェクトの接続情報と許可するメールアドレス（\`ALLOWED_EMAILS\`）を設定。
3. \`firebase deploy --only hosting\` でデプロイ。

## 📱 iOS・Chromeの認証対応について
iOSのChromeブラウザや、ホーム画面に追加したPWA等の環境下において発生する「セッション破棄・リダイレクト後のログイン失敗（ITP）」問題への対策として、アクセス環境（iOSかどうか/PWAかどうか）を自動判定し、\`signInWithPopup\` と \`signInWithRedirect\` を適切に切り替える実装を導入しています。
